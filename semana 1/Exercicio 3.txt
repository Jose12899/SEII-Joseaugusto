/bin - Contém binários e executáveis ​​importantes que podem ser acessados ​​a qualquer momento a partir do terminal chamando seus nomes. Esses arquivos incluem, por exemplo , vim, gzip, gunzip, tar.  
 /sbin - Contém os binários e executáveis ​​mais importantes para root.  
 /lib - Contém código compartilhado entre binários

 -- ->/bin e /sbin Contém binários e programas executáveis ​​que não são exigidos pelo sistema operacional e são destinados ao uso por usuários finais. -- ->/local - Contém binários compilados manualmente pelo usuário para que não entrem em conflito com outros binários. -- ->/etc -Contém arquivos de configuração editáveis ​​( definição de texto editável)  
 -- ->/home - Contém os diretórios home de cada usuário ----> Inside / at home  
 -- -- -> /username – contém pastas com os arquivos pessoais do usuário. $PATH - é um arquivo que mapeia a localização de todos os binários a serem executados no terminal. which - o comando localiza o arquivo binário.  
 /boot - contém os arquivos necessários para inicializar o sistema.  
 /dev - Contém arquivos de dispositivo que se conectam e permitem a comunicação com o sistema. /opt - Contém software adicional.  
 /var - Contém arquivos de variáveis ​​dinâmicas comoregistros e cache. /tmp - Contém arquivos temporários que não são salvos na inicialização.  
 /proc - Este é um diretório fora do disco que é carregado na memória para que o kernel do Linux acompanhe os processos em execução.
